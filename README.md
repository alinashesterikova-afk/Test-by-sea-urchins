# Local prototype

## Open locally

1. Open a terminal in this folder:
   `/Users/farben/Documents/Prototype`
2. Start the local server:
   `npm start`
3. Open this URL in the browser:
   `http://127.0.0.1:3000/`

## Video file

The prototype looks for a local file named `Video` with a supported extension, for example:

- `Video.mp4`
- `Video.webm`
- `Video.mov`

Place it in this folder, next to `index.html`.

## Share with colleagues

Options:

1. Send them the folder path and ask them to run `npm start`, then open `http://127.0.0.1:3000/`.
2. Zip the whole folder and send the archive, including the video file.
3. Put the folder in Git and share the repository with a short setup note.
